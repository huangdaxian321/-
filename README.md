# 贪吃蛇游戏

这个项目提供了一个使用 Pygame 编写的贪吃蛇游戏示例，包含以下特性：

- 玩家控制的蛇与 AI 蛇对战，两条蛇都可以吃食物并变长；
- 计分与关卡系统，玩家得分提升会增加速度并生成随机障碍物；
- 玩家蛇撞到边界、障碍物、自己或 AI 蛇时游戏结束；
- AI 蛇撞墙或撞到障碍物、玩家蛇时会死亡并在随机位置复活，长度重置为 1，得分归零；
- 计时限制为 2 分钟，剩余时间会在屏幕上方显示；
- 游戏启动后会先显示简短介绍，然后自动开始；
- 游戏结束后呈现特效并停留，关闭窗口即可退出；
- AI 每一步有约 5% 的几率做出随机方向选择，动作更具不确定性。

运行方式：

```bash
pip install pygame
python snake_game.py
```

启动后使用方向键控制玩家蛇移动，屏幕上方显示玩家和 AI 的得分及关卡。
